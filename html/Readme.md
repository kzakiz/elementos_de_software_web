Solución:

1. a. Servidor DNS:
Traduce nombres de dominio a direcciones IP.

b. IP:
Número único que identifica un dispositivo en Internet.

c. Protocolo:
Conjunto de reglas para la comunicación entre dispositivos.

d. Nombre de dominio:
Nombre fácil de recordar que representa una IP (ej: google.com).

e. Directorio y archivo:
Ruta dentro de una web donde está un recurso.

Directorio: carpeta
Archivo: documento específico

2. Roles

a. Diseñador UI/UX:
Diseña la apariencia y la experiencia del usuario.

b. Cloud Manager:
Administra servidores y servicios en la nube.

c. Administrador de base de datos:
Gestiona, protege y organiza los datos.

d. Project Manager:
Planifica, coordina y supervisa el proyecto.
